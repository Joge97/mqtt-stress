# mqtt-stress

> CLI Tool for sending a lot of messages over mqtt for stress testing services

## Install

Install with [npm](https://www.npmjs.com/)

```sh
$ npm i mqtt-stress --save-dev
```

## Usage

```js
var mqtt-stress = require('mqtt-stress');
```

## Running tests

Install dev dependencies:

```sh
$ npm i -d && npm test
```
